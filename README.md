# موقع مشاركة الموقع مع بوت التليغرام

موقع بسيط يتيح للمستخدمين تحديد موقعهم الحالي على الخريطة وإرساله إلى بوت تليغرام.

## المميزات
- 🗺️ عرض الموقع على خريطة تفاعلية
- 📍 تحديد دقيق للموقع باستخدام GPS
- 📤 إرسال الموقع إلى بوت تليغرام
- 🌐 الحصول على العنوان التقريبي من الإحداثيات
- 📱 تصميم متجاوب مع جميع الأجهزة

## طريقة الإعداد

### 1. إنشاء بوت تليغرام
1. افتح تطبيق تليغرام وابحث عن `@BotFather`
2. أرسل الأمر `/newbot` واتبع التعليمات
3. احفظ التوكن الذي ستحصل عليه

### 2. الحصول على Chat ID
1. أرسل رسالة إلى البوت الجديد
2. افتح الرابط التالي في المتصفح (استبدل YOUR_BOT_TOKEN بالتوكن الخاص بك):
   `https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates`
3. ابحث عن `chat` وانسخ الرقم الموجود في `id`

### 3. إعداد الموقع
1. افتح ملف `config.js`
2. ضع توكن البوت في `TELEGRAM_BOT_TOKEN`
3. ضع Chat ID في `TELEGRAM_CHAT_ID`

### 4. تشغيل الموقع
- يمكنك تشغيل الموقع محلياً أو رفعه على أي استضافة ويب
- تأكد من تفعيل HTTPS إذا كنت تريد استخدام الموقع على الإنترنت (لأن تحديد الموقع يتطلب HTTPS في بعض المتصفحات)
